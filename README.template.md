# Observable

A library that provides classes that implement the Observer Pattern.

## Installation

### Deno

```ts
deno install jsr:@mishieck/observable
```

### NPM

```ts
npx jsr install @mishieck/observable 
```

### Bun

```ts
bun x jsr install @mishieck/observable 
```
